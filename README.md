# mymvc
This is simple MVC using PHP, Eloquent, Bootstrap

# Configuration
1. Change RewriteBase .htaccess on root/public
2. Change ROOT_DIR and PUBLIC_DIR to your folder

# Database

3. 'username'=>'root',
